Kingsoft Kuaipan Client

Kingsoft Kuaipan is a Linux cloud storage client jointly developed by Ubuntu Kylin Team and the Kingsoft Cloud Corp.. It provides file synchronization, backup, and sharing capabilities. You can manage your files across all OS now! You are welcome to install it via Ubuntu Kylin Software Center or www.ubuntukylin.com.
