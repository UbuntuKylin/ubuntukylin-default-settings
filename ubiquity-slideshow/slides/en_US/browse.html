Sogou Input Method

Sogou Input Method for Linux is co-developed by Ubuntu Kylin team and Sogou Corp.. It is based on Fcitx framework and supports the same user experience with Windows, Mac version. We will rlease the first formal version in Apr 2014. You are welcome to install it via Ubuntu Kylin Software Center or www.ubuntukylin.com
